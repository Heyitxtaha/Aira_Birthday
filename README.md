This is a personalized and interactive birthday celebration website created with HTML, CSS, and JavaScript (GSAP library for animations). It's designed to bring a smile to Aira's face on her special day!

## Features

* **Interactive Landing Page:** A welcoming page with floating emojis and a hidden message that reveals an "Enter My Day" button and a GIF upon interaction.
* **Animated Birthday Wish:** A dynamic "Happy Birthday, Aira <3" text animation with twinkling stars, falling rose petals, and heart-shaped fireworks.
* **Clickable Birthday Cake:** A dedicated page where clicking the cake extinguishes the candle and triggers a shower of ribbons, revealing a "Next Page" button.
* **Personalized Birthday Card:** A digital card with a custom message for Aira, featuring interactive elements and animated decorations.
* **Responsive Design:** Adapts to different screen sizes for a consistent experience on desktop and mobile.

## Technologies Used

* **HTML5:** Structure of the web pages.
* **CSS3:** Styling and animations.
* **JavaScript (ES6+):** Interactive elements and canvas animations.
* **GSAP (GreenSock Animation Platform):** For powerful and smooth JavaScript animations.
* **Tailwind CSS (CDN):** Used on the initial landing page for utility-first styling.

## Customization

You can easily customize this project for another person:

1.  **Change Names:**
    * In `index.html`, update the button text to `<span>Celebrate [New Name]'s Birthday &lt;3</span>`.
    * In `birthday.html`, change the `h1` content to `Happy Birthday, [New Name] <3`.
    * In `final.html`, update the `<p>Dear Muqadas,</p>` to `<p>Dear [New Name],</p>`.
2.  **Update Messages:** Edit the `<p class="romantic-message">` in `birthday.html` and the wishes in `final.html`.
3.  **Replace Images/GIFs:** Update the `src` attributes of `<img>` tags in the HTML files and replace the image files in the `images/` directory.
